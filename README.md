## Prerequisites
MERN=>mongodb, express, react, nodejs
## Quick locally run
```
1. git clone https://github.com/prevision-soft/online-shop.git
2. cd online-shop
3. npm install
4. cd client
5. npm install
6. cd ..
7. npm run dev
```
